# Aviantur-BackEnd

## Instalación
```
> git clone https://github.com/DevLucho/Aviatur-BackEnd.git
> cd Aviatur-BackEnd-master
> npm install
```

## Servidor
```
> npm run dev

```
- http://localhost:8080/

## Configuración DB
- MongoDB
- Configurar DB en el archivo .env (nombreDB, puerto, host) 
- Importar datos: sample/data.json
- Correr servidor
